# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-23

### Added

- 10-question science-based chronotype quiz
- 4 detailed result profiles — Lion, Bear, Wolf, Dolphin
- Score breakdown visualization across all chronotypes
- Interactive daily timeline per chronotype (mobile + desktop)
- 5 personalized productivity and sleep tips per result
- Dark cosmic theme with animated star background
- Mobile-first responsive design
- Bilingual support — English and Korean (i18next)
- Social sharing — Twitter, Facebook, KakaoTalk, Copy Link
- Serverless backend — AWS Lambda + API Gateway + PostgreSQL
- Global statistics — chronotype distribution across all users
- SEO optimization with JSON-LD structured data
- Analyzing overlay animation between quiz and results
